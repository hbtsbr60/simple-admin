# Simple Server Admin

> Admin content management system

## Prerequisites

- [Yarn](https://yarnpkg.com/)

## Tools

- [React](https://reactjs.org/)
- [Material UI](https://mui.com/)
- [Apollo GraphQL client](https://www.apollographql.com/docs/react/)

## Start

```sh
yarn start
```

## Build

```sh
yarn build
```
